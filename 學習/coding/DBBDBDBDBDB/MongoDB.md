#### Shell 指令

- `mongosh` 在CMD開啟mongo shell
- `cls` 清除
- `exit` 離開
- `show dbs` 顯示所有DB
- `show collection` 顯示該DB下的所有collection
- `use (DB_name)` 進入該DB的操作區
// 該DB可以是不存在的 一但新增資料 芒果會幫你自動新建DB
