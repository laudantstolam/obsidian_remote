### 佈署three.js
1. CDN
`  <script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/0.157.0/three.min.js"></script>`
2. 或是直接安裝
`git clone https://github.com/mrdoob/three.js.git`

### 佈署框架
1. vite
`npm start vite@latest`
2. 選擇框架
`vue/react/......etc`
3. 安裝套件
`npm install`
`npm install three gsap dev`
4. 開啟伺服器
`npm run dev`
